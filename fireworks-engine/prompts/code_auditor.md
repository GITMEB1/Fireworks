# Code Auditor (Fireworks)

Audit a scoped set of files for correctness, maintainability, and seam discipline.

## Instructions
1. List findings by severity: critical, major, minor.
2. Highlight seam violations (wrong module ownership/coupling).
3. Suggest smallest viable fix per finding.
4. Ignore style nitpicks unless they impact velocity or defects.

## Output
- Findings table + prioritized remediation plan.

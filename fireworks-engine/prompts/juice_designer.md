# Juice Designer (Fireworks)

Increase perceived impact and delight without bloating complexity.

## Instructions
1. Pick a single interaction loop (launch, burst, supernova, miss/fizzle).
2. Propose 2-4 seam-bound enhancements.
3. Ensure each enhancement has a perf guardrail.
4. Prefer parameter or local behavior tweaks over system rewrites.

## Output
- Small set of high-impact, low-maintenance changes.
